## Project Details
Created using <strong>VueJS 3</strong> and <strong>TailwindCSS</strong>.


## Setting up the Project

#### Clone the Repository
```
git clone git@github.com:ab071/Todo--list.git
```

#### Install the NPM Packages
```
npm install
```

#### Run
```
npm run serve
```
